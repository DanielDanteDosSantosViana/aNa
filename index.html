aNa by joeldossantos

aNa - API NCL para Autoria possui um conjunto de classes que representam os elementos da linguagem NCL (Nested Context Language).

Cada elemento contém um conjunto de funções que permitem a criação de um documento através de códigos Java, bem como a criação da estrutura de dados provida pela API a partir de um arquivo XML que especifique um documento NCL. Além da representação dos elementos, a aNa também provê a possibilidade de verificação da integridade do documento NCL através de funções de validação.
